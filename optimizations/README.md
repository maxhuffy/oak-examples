# Optimizations

This section contains runnable scrips that serve to be used as backed for benchmarking frontend optimizations.

## Running Examples

The examples in this folder run on your host computer, communicating with the OAK device. Follow these steps to run an example:

1. **Install dependencies**: Navigate to the example folder and install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

2. **Ensure good network bandwidth**: Make sure you have good network bandwidth (> 1Gbps) between the device and computer

3. **Run the example**: Execute the Python script:
   ```bash
   python3 main.py
   ```
